# Plugins

Let's start with a short list of my note taking related plugins that implement a specific feature and can be combined.

  • Markdown All in Oneopen in new window: Markdown language support
  • markdown lintopen in new window: Markdown linter tool
  • Prettieropen in new window: multi-language formatting tool, very good support for markdown
  • PicGoopen in new window: image upload (this is a third-party program)
  • Markdown PDFopen in new window: Markdown export PDF
  • Draw.io Integrationopen in new window: draw.io Flowchart Integration
    • Markdown Preview Mermaid Supportopen in new window: mermaid flowchart support DSL, but draw.io is better
  • vscode-mindmapopen in new window: mind map integration (this is the fork version, but it's much better)
  • Local Historyopen in new window: local history
  • Code Spell Checkeropen in new window: Word spell checker
  • joplin-vscode-pluginopen in new window: joplin notes integration

You may notice that some of these plugins have nothing to do with markdown, but they are still important for editing, for example the spell checker feature that was only recently implemented in Joplin (v1.4.*) already exists in the existing VSCode plugins.
