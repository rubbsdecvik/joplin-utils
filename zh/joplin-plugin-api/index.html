# joplin-plugin-api

# 简介

封装 joplin-plugin 的类型定义以及暴露出来的 enum 之类的，单独作为一个 npm 包维护便于复制 joplin plugin 的那些 ts 文件。

# FAQ

# 和 @thejohnfreeman/joplin-plugin 有什么不同？

这个包是 joplin-plugin-api 的灵感来源，但它将 joplin plugin 的类型定义与打包功能混合到一个包了，而这应该是两个包的。
